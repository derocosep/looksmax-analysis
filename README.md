# 🚀 LooksMax AI Analysis Bot

Телеграм-бот на базе искусственного интеллекта для анализа внешности и предоставления персональных рекомендаций по луксмаксингу. 

---

## ✨ Основные функции
- 🧠 **AI Анализ**: Обработка данных пользователя для выявления зон роста.
- 📋 **Персональные гайды**: Генерация пошаговых инструкций по уходу (skin care), стилю и грумингу.
- 💬 **Интерактивный чат**: Ответы на вопросы в режиме реального времени.
- 💳 **Интеграция платежей**: Поддержка оплаты через **TON** (Tonkeeper) для премиум-функций.

## 🛠 Технологии
- **Runtime**: [Node.js](https://nodejs.org/)
- **библиотека Bot API**: Telegraf / GrammY
- **AI**: OpenAI API (или другое решение)
- **Database**: MongoDB / PostgreSQL (если используешь)
